# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현 기능 목록
- 자동차 생성
- 자동차 이름 부여
- 사용자의 게임 초기화( 몇 번을 이동할 것인지 )
- 자동차 전진 기능
- 우승자 선택 기능
